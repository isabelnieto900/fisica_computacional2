# Proyecto Tiro Parabólico

Este proyecto simula el movimiento parabólico de un objeto, guarda los resultados en un archivo `.dat` y permite graficarlos con `gnuplot` o `python`.

## Compilación

```bash
make
```

## Ejecución y Graficación

```bash
make plot
```

## Limpieza

```bash
make clean
```
